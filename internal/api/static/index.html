The Problem

Every agent team hits the same wall

Your agent gets blocked

Cloudflare, PerimeterX, and DataDome catch every headless browser. Your Playwright script worked yesterday. Today it returns a CAPTCHA page.

Your agent goes rogue

No spending limits. No domain restrictions. No human approval for sensitive actions. One misconfigured prompt and your agent is buying things, accepting ToS, or scraping banned sites.

Your agent returns pixels, not data

Screenshots and raw HTML are not structured data. Your agent needs typed JSON — product prices, form fields, search results — not a 2MB PNG to feed back to the LLM.

The Solution

Three layers that make agents production-ready

Solves: Agent gets blocked

Stealth Engine

uTLS proxy rewrites TLS ClientHello to match real Chrome fingerprints. SwiftShader crowd-blends WebGL across all instances — identical output, no random noise to detect. Bézier mouse humanization. Three auto-detected modes: research, hardened, max.

{"stealth": "pass", "tls": "chrome_122",
 "webgl": "swiftshader", "detected": false}

uTLS + SwiftShader · JA4 Clean · 3 Modes

Solves: Agent goes rogue

xBPP Governance

Every action policy-checked before execution. Domain allowlists, action type restrictions, per-session budgets, rate limiting, transaction thresholds with human escalation. Import authenticated sessions with trust_mode: preserve — your LinkedIn session fingerprint stays intact.

{"policy": "prod-agent", "checks_passed": 12,
 "trust_mode": "preserve", "session": "imported"}

12 Policy Checks · Session Import · Trust Preserve

Solves: Agent returns pixels

Stateful Task Planner

Send one goal. The planner decomposes it into steps, handles pagination, accumulates data across pages, and re-plans when something unexpected happens. SSE streaming shows progress in real-time. Resume from checkpoint if it crashes.

{"step": 4, "total": 9, "extracted": 47,
 "page": "2/3", "status": "paginating"}

Multi-Page · SSE Streaming · Auto Re-Plan

Developer Experience

Built for developers who ship

TypeScript SDK

npm install @aperture/sdk

Fluent API with full type safety. Zod-validated responses. Works with any agent framework.

import { Aperture } from '@aperture/sdk'

const client = new Aperture({ apiKey: 'ap_...' })

// Multi-page task with SSE streaming
const stream = await client
  .task('Extract LinkedIn connections pages 1-3')
  .withMode('hardened')
  .stream()

for await (const event of stream) {
  console.log(event.type, event.extracted?.length)
}
// progress 0  →  data 10  →  data 20  →  complete 30
